# Maps Search Link
Adds a Maps link to the Google Search navigation bar.

Made for, and tested with, [Userscripts for Safari](https://apps.apple.com/app/userscripts/id1463298887). May work with other similar extensions.

## Configuration
If Google Search shows a tab labelled something other than "Alle" in your language (e.g. "All" in English), update the `ALL_TAB_LABEL` variable at the top of the script to match.
